# Thai Food Classifier

A monorepo for the Thai food image classification system — an AI-powered web application that identifies Thai dishes from photos.

## Structure

```
thai-food-classifier/
├── apps/
│   ├── web/       # Next.js frontend
│   └── api/       # Flask API backend
├── packages/
│   └── shared/    # Shared food labels (JSON, Python, TypeScript)
└── infra/         # Docker Compose and Dockerfiles
```
